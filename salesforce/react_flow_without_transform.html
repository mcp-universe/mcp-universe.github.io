âš¡ Key Features of Non-Transform Mode

  • Direct Output: Tool responses are used directly without modification
  • Faster Execution: No additional LLM calls for transformation
  • Raw Data Preservation: Complete tool output is preserved in agent history
  • Optional Summarization: Can use LLM summarization instead of transformation
  • Simpler Flow: Fewer decision points and error handling paths
  • Lower Token Usage: No additional prompts for transformation logic
